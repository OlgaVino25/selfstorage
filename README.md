# SelfStorage

SelfStorage — это веб-платформа для аренды складских боксов индивидуального хранения. Проект позволяет пользователям просматривать доступные склады и боксы, оформлять аренду, применять промокоды, а также получать уведомления о приближающемся окончании срока хранения. Для администраторов предусмотрен удобный интерфейс управления складами, боксами, заказами и контентом (FAQ, отзывы). Также реализован собственный сервис коротких ссылок для отслеживания эффективности рекламных кампаний.

## Основные функции

- Регистрация и авторизация пользователей (с расширенным профилем: телефон, адрес, аватар)
- Просмотр списка складов и доступных боксов с фильтрацией по размеру
- Расчёт стоимости аренды с учётом промокодов
- Оформление заказа (с опциями доставки и замеров)
- Личный кабинет пользователя с историей аренд
- Управление промокодами в админке
- FAQ с категориями и WYSIWYG-редактором (TinyMCE)
- Отзывы с возможностью загрузки фото и превью в админке
- Собственная система коротких ссылок для рекламных кампаний (учёт переходов)
- Ежедневная email-рассылка с напоминаниями об истечении срока аренды (APScheduler)

## Используемые технологии

- **Python 3.13+**, **Django 5.2**
- **SQLite** (по умолчанию, можно заменить на PostgreSQL/MySQL)
- **Bootstrap 5** – фронтенд
- **TinyMCE** – редактор для FAQ
- **APScheduler** – планировщик задач для email-уведомлений
- **environs** – управление переменными окружения

## Установка и запуск

1. Клонируйте репозиторий
2. Создание и активация виртуального окружения

```bash
# Установка uv (если ещё не установлен)
pip install uv

# Создание виртуального окружения и установка зависимостей
uv venv
source .venv/bin/activate           # Linux/macOS
.\.venv\Scripts\Activate.ps1         # Windows
uv sync
```

Это использует `pyproject.toml` и автоматически синхронизирует окружение.

3. Настройка переменных окружения

Создайте файл .env в корне проекта и укажите следующие переменные:

```env
SECRET_KEY=your-secret-key
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1

# Настройки email (для уведомлений)
EMAIL_HOST=smtp.yandex.ru
EMAIL_PORT=465
EMAIL_HOST_USER=your_email@yandex.ru
EMAIL_HOST_PASSWORD=your_app_password
DEFAULT_FROM_EMAIL=SelfStorage <your_email@yandex.ru>
```

_Важно: Для работы email через Яндекс необходимо создать пароль приложения._

4. Применение миграций

```bash
python manage.py migrate
```

5. Создание суперпользователя

```bash
python manage.py createsuperuser
```

6. Запуск сервера разработки

```bash
python manage.py runserver
```

_Сайт будет доступен по адресу http://127.0.0.1:8000, админка – http://127.0.0.1:8000/admin._

## Дополнительные команды

### Запуск планировщика уведомлений

Для ежедневной рассылки напоминаний об окончании аренды выполните:

```bash
python manage.py notify_rentals
```

Команда запускает фоновый процесс, который будет проверять даты и отправлять письма каждый день в 10:00.

### Короткие ссылки для рекламы

В приложении `shortlinks` реализована простая система учёта кликов:

- В админке создаётся запись с уникальным slug (например, tg1) и целевым URL.
- Ссылка вида `https://вашсайт.ru/go/tg1/` при переходе увеличивает счётчик `clicks` и редиректит на целевой адрес.
- В списке админки отображается полная короткая ссылка с возможностью перехода.

Это позволяет оценить эффективность разных рекламных каналов без использования сторонних сервисов.

### Структура проекта

Проект состоит из нескольких Django-приложений:

**accounts** – расширение модели пользователя (профиль с телефоном, адресом, аватаром), регистрация и логин.
**main** – главная страница, страница FAQ (категории и вопросы/ответы), отзывы.
**promotions** – управление промокодами.
**rent** – аренда: создание заказа, применение промокодов, хранение списка вещей.
**selfstorage** – основной пакет проекта (настройки, корневые URL).
**shortlinks** – собственный сервис коротких ссылок для отслеживания переходов из рекламных каналов.
**warehouses** – модели складов и боксов, расчёт стоимости, отображение доступности.
