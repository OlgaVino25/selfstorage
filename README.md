# SelfStorage

Реализованы модели данных и интерфейс администратора.

## Требования

- Python 3.13+
- [uv](https://docs.astral.sh/uv/) (рекомендуемый менеджер пакетов)


1. Установите зависимости с помощью uv

```bash
uv sync
```

Будут установлены все пакеты из pyproject.toml.

2. Создайте файл `.env` в корне проекта
3. Создайте папку `media/` в корне проекта
4. Примените миграции:

```bash
uv run manage.py migrate
```

5. Создайте суперпользователя для доступа в админку:

```bash
uv run manage.py createsuperuser
```

6. Запустите сервер разработки:

```bash
uv run manage.py runserver
```

Откройте http://127.0.0.1:8000/ в браузере.
Админка доступна по адресу http://127.0.0.1:8000/admin/.

Структура проекта:

```text
accounts          Профили зарегистрированных пользователей (расширение User)
client            Контакты незарегистрированных клиентов (для консультаций)
consultation      Записи на консультации
main              FAQ и отзывы
media             Создать папку в корне (она не включена в репозиторий)
promotions        Промокоды
rent              Аренда боксов
selfstorage       основная директория с настройками Django
services          Дополнительные услуги (например, аренда стеллажей)
static            Создать папку в корне. Создать 2 подпапки css и img и положить в них из zip от Егора
templates         HTML-шаблоны страниц
warehouses        Склады и боксы
```
